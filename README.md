# seahorse

<!-- badges: start -->
<!-- badges: end -->

This package provides tools to facilitate the analysis of Seahorse
bioenergetic data in R.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("wmoldham/seahorse")
```
